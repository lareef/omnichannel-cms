How Escalation Targets Work

Specific User

Escalate to a specific person (e.g., supervisor, manager)

Role-Based

Escalate to all users with a specific role (e.g., all supervisors)

Department

Escalate to an entire department (e.g., customer service department)

Target Order Matters

Targets are processed in order (1, 2, 3, etc.). You can create multiple targets for the same policy level. For example: first notify the supervisor, then the department head, then management.
